== Description ==

Allows you to post content on multiple WordPress sites from one central location.

= Benefits / Features =
* Minimal elements on a page so it loads very quickly. It is based on Bootstrap (it intentionally does not load within WP Admin).
* Publish or Save posts as drafts all managed from one single location
* Remembers the settings -> publish post, new lines, last used site.
* Remembers logins of multiple sites
* Simple preview
* The page does not refresh (all done through Ajax)

== Installation ==

= Automatic Install =
Please go to Wordpress Admin &gt; Plugins &gt; Add New Plugin &gt; Search for: Orbisius Just Write and then press install

= Manual Installation =
1. Upload orbisius-bbpress-enhancer.zip into to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
